Файлы формата .bat расположены по адресу ..\HW5-Frolova_E_D-BSE185_1\out\artifacts\HW5_Frolova_E_D_BSE185_1_jar
По кнопке Multiplay игра переходит в многопользовательский режим (2 игрока).
**********Расстановка**************
Корабли расставляются при помощи кнопок, расположенных в верхнем правом углу.
По наведении на кнопку, подождав 2 секунды можно узнать информацию о корабле (длина, кол-во нераставленных кораблей этого типа)
Для того, чтобы повернуть корабль на 90 градусов нужно нажать кнопку CTRL.
Сбросить расстановку можно при помощи кнопки Remove All.
Кнопка Play обеспечивает подключение клиента и сервера.
Для того, чтобы кнопка стала кликабельной, необходимо расставить все корабли.
**********Процесс игры*************
После ввода запрашиваемых данных появляется окно с сообщением - именем противника.
Левое поле показывает расстановку кораблей игрока, правое - противника.
При переходе хода к игроку, его имя выделяется красным цветом. 
При переходе хода к противнику, красным цветом выделяется имя противника.
Справа ведётся панель логгирования.
Под панелью логгирования расположена кнопка преждевременного выхода из игры.
**********Завершение игры**********
Игра завершается, когда один из участников убивает все корабли,
либо после нажатия кнопки Exit.
После чего появляется оповещение о конце игры с информацией об участнике,
вызвавшим данное действие.
После закрытия данного окна, появляется информация о количестве ходов, 
после чего программа завершает своё исполнение.
